如何編輯CMSimfly

1.開啟隨身系統 P37 中的 start.bat

2.於 dos 環境中進入至 site-s40523125 的資料夾

3.進入資料夾後執行 python wsgi.py 指令後 於近端開啟

4.開啟近端網頁後輸入密碼 admin 登入後進行編輯

5.編輯完成後回到 dos 環境輸入 git add . 確認編輯內容

6.輸入 git commit -m "xxx" 確認版本名稱

7.最後輸入 git push  並輸入帳號密碼即完成編輯流程
